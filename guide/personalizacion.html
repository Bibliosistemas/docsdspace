# La personalización de Dspace 7

La personalización de Dspace lleva necesariamente una serie de pasos asociados al desarrollo de Angular (opens new window)

Para iniciar el desarrollo se requieren de algunos programas y configuraciones.

  • nodejs (opens new window)
  • angular
  • visualCode (opens new window) o un editor de código
  • yarn (opens new window)
  • git (opens new window)
