# Proceso de instalación entorno en local

# Requerimientos

Tener instalado nodejs, yarn, git y VisualCode (como editor) Instalación de Angular (global)

npm install -g @angular/cli

Clonar el repositorio Crear directorio de trabajo

$ mkdir d73 
$ cd d73 
$ git clone https://github.com/DSpace/dspace-angular.git

Instalar dependencias

$ yarn install

Arrancar el server de desarrollo de Angular


$ ng serve --host 0.0.0.0
